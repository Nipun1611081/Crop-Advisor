About Us

In order to mitigate the agrarian crisis in the current status quo, there is a need for better recommendation systems to alleviate the crisis by helping the farmers to take an informed decision before starting the cultivation of crops. The “Crop Advisor for Crop Cultivation using Machine Learning and Neural Network approach” would be an application that will suggest suitable crops to be cultivated based on various factors such as rainfall, temperature, soil type etc.
